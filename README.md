# REACH Center Workshop April 8 from 11:00am - 1:00pm ET
## Global Health in Context: Linking Census Data to Health Surveys using IPUMS DHS and IHGIS

This repository contains supporting materials for the hands-on exercerise portion of the workshop with the Research and Engagement for Action on Climate and Health (REACH) Center on linking contextual data from IPUMS IHGIS with health data from IPUMS DHS. This workshop is held in a hybird format: in person at the Milken Institute School of Public Health (6th Floor Conference room), or via Zoom. 

This workshop is lead by IPUMS scientists - Dr. Miriam King, and Tsu Zhu - introducing freely available data from: Demographic and Health Surveys (IPUMS DHS) -harmonized individual records from DHS program surveys conducted around the world - and International Historical Geographic Information System (IPUMS IHGIS) - consistently formatted tables of population and agricultural censuses around the world.

### Files contained in this repository:
- Word Doc with step-by-step instructions on how to download IPUMS DHS and IPUMS IHGIS data as well as the census-to-DHS crosswalks (CSV files) to answer a sample research question
- R script that walks through data preparation, exploration, and analysis for this research question
- PDF slides with screenshots from IPUMS DHS and IPUMS IHGIS websites as a visual aid on how to get data
- REACH event flyer
- REACH workshop overview that contains instructions for data access and other event details
